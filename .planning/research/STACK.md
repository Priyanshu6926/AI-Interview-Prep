# Research: Stack

## Recommended Stack for Voice AI Interviews
- **Voice Agent Orchestration**: Vapi.ai (industry standard for WebRTC and LLM low-latency orchestration).
- **Core LLM**: Google Gemini 1.5 Pro / Flash (has strong multimodal capability, currently supported by Vapi).
- **Backend Framework**: Node.js/Express (existing).
- **Frontend Framework**: React + Vite (existing).
- **Database**: MongoDB (existing).
